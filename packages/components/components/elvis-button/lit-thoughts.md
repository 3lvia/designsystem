# Lit

Pros:

- Liten bundle size
- Enklere arkitektur
- "Vanilla" teknologi
- Raskere utvikling (kan fjerne Gulp)
- Shadow DOM
- Åpner for åpen DOM 😍

Cons:

- Ny teknologi (enda mer å lære)
- Sprer teknologien (flere byggsteg)
- Mindre verktøykasse (ikke portal bygd inn)
- Ikke SCSS?
- Kan ikke bruke eksisterende toolbox (må lage en for Lit)
