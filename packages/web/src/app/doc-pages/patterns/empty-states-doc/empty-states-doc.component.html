 A good empty state should explain what's going on in plain language. Tell the user what the next step is and guide them forward with a button or link if needed. Don't use product-specific terms that the user may not understand. Use a positive tone, motivate the user and add an emoji in the message to express personality. Use graphical elements such as illustrations, colors, and icons to support the information. You can use a neutral or humorous tone to it, but try to play with our concept OFF/ON - most often this is a great opportunity for when something is off!
