The message, illustration, typography, button/link and layout all depends on where the empty state should appear, what the state signalises and what actions are required. Here are some examples of different cases.

Messages that are displayed on the entire page should use larger typographies and the illustration should scale accordingly. When the message is related to only parts of the content the illustration and typography sizes should be adjusted to fit the surrounding content. This is just some examples, adjust the sizes to your specific case. The illustration can be aligned to the left if it fits the layout better. However on mobile the vertical layout should be used to make better use of the space.
