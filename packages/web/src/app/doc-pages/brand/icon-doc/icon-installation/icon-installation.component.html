1. Install packages
    npm install @elvia/elvis-icons @elvia/elvis-assets-icons or yarn add @elvia/elvis-icons @elvia/elvis-assets-icons
  2. Register icons

    All icons you wish to use must be registered in the global icon registry. This is done by importing the icons from @elvia/elvis-assets-icons, and then registering them with the addIcons-function from the @elvia/elvis-icons package.

    NB! Each icon only needs to be registered once, and can then be used in multiple places. This should be done in a global file, such as main.ts.
