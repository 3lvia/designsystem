To apply a custom color, you can overwrite the CSS-variables used in the icon. The variables used are typically:

  • --e-color-icon-stroke-1
  • --e-color-icon-filled-background-1
  • --e-color-icon-filled-foreground-1

To see what specific variable is used for an icon, inspect the icon in the browser.
