Red Hat Display has denser letter space and higher x-height and is suitable in large sizes, for example in titles.

Red Hat Text has a slightly smaller x-height, more space and is designed for good readability in smaller sizes.

Red Hat Mono has characters that take the same amount of horizontal space. Mono are used to make it easier to comparing numbers and to distinguish code from natural-language text.

You can download all the fonts via Google Fonts .

We use Verdana as an alternative system font. It’s only used in the cases where it isn’t possible to use the profile font, for example digital documents and templates that are shared with people outside of Elvia who might not have the profile font installed. Verdana is less dense than Red Hat, but to simulate Red Hat in the best way you can pinch the text by reducing the letter spacing. Red Hat Display Black should be used in all titles. Titles should be clear and easy to recognize. As a rule, they should be uppercase, but in the case of longer texts, lower case letters should be used to ensure good readability.

Brand titles can be activated to show that Elvia is ON and fully charged. To do this, we use text with outline (OFF) in combination with filled text (ON) and our green dot at the end (fully charged).

The activated text should always be filled in solid black or white, depending on which gives best contrast to the background. The deactivated text should have an stroke width (black or white, depending on the background), with a thickness following the following formulas:

Pixel size of the text x 0.02 = line thickness on the outline measured in pixels. Point size of the text x 0.017 = line thickness on the outline measured in points.
Can be written in capitals or lower case. The entire text can be outlined. If parts of the text are to be highlighted, the highlighted part can be set as filled. We mainly use Red Hat Display Bold in titles. Use Red Hat Text Medium, and should always be smaller than the title. Ingress should have a larger point size than the body text, to create a hierarchical relationship between them. Must always be set in an easily readable size. This text should have a smaller point size than the title and preamble. We mainly use Red Hat Text Regular in body text, but we can emphasise important content with italic, underlined or bold. Always set in bold, and is 0-2 points larger than the body text that follows. We end titles with a green full stop.

All typography can be used through a set of classes. The classes set multiple attributes, like font-size, line-height and font-weight. These are all available when using Elvis. All of our typography classes are listed at the top of this page.
