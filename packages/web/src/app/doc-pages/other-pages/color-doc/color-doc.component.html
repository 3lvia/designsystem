All colors should be used in solid 100%, with the exception of illustrations, graphs and labels. In these cases, color shades can be used as long as the color isn’t the only visual indication of important data. Alle farger skal brukes i heldekkende 100 %, med unntak av illustrasjoner, grafer og etiketter. I disse tilfellene kan fargenyanser brukes så lenge fargen ikke er den eneste visuelle indikasjonen på viktige data. Elvia is mainly white and dark grey, with some black components and a touch of green in details such as icons, illustration and active states. Elvia er hovedsakelig hvit og mørkegrå, med noen svarte komponenter og et snev av grønt i detaljer som ikoner, illustrasjoner og aktive tilstander.
Green shows that Elvia is ON, activated, fully charged and environmentally friendly. The green color should be used intentionally and should not be used on large surfaces as a background. The green color are used to display when something is active/clickable on interactions. It’s also used in illustration and icons to highlight the most important or just to get a touch of the green light in between all the white and black. Grønn viser at Elvia er PÅ, aktivert, fulladet og miljøvennlig. Den grønne fargen bør brukes bevisst og bør ikke brukes på store flater som bakgrunn. Den grønne fargen brukes til å vise når noe er aktivt/klikkbart på interaksjoner. Den brukes også i illustrasjoner og ikoner for å fremheve det viktigste eller bare for å få et snev av grønt lys mellom alt det hvite og svarte. White stands for light and energy. White is used as a background colour and as a text colour on darker surfaces. Hvitt står for lys og energi. Hvitt brukes som bakgrunnsfarge og som tekstfarge på mørkere flater. Black shows that Elvia is OFF. It’s the primary color for text and icons, but can also be used to get a dark surface. Svart viser at Elvia er AV. Det er den primære fargen for tekst og ikoner, men kan også brukes for å få en mørk overflate. Dark (grey) is the preferred dark background. Mørk (grå) er den foretrukne mørke bakgrunnen. Elvia colors are included in @elvia/elvis as CSS classes and variables. All colors can be used through a set of classes. The e-text sets the text-color of the element, while the e-bg sets the background as well as the text.
  • e-bg-name
  • e-text-name

All bg-classes sets the text-color to the contrast-color of the background. It is therefore not necessary to use both the bg and the text class on the same element, exceptions can of course occur and then the text class is available.

All colors can be used through their CSS-variable as well. To use the CSS-variables you need to install the ponyfill css-vars-ponyfill. Read more about using CSS variables

  • var(--e-name)
