Typography
Elvia has a profile font called Red Hat that should be used throughout all material. The Red Hat font family inlcudes two optical sizes: Display and Text, and can be dowloaded here dowloaded here via Google Fonts.
All typography can be used through a set of classes. The font-classes set multiple attributes, like font-size, line-height and font-weight.
Red Hat Display has denser letter space and higher x-height and is suitable in large sizes, for example in titles. Red Hat Text has a slightly smaller x-height, more space and is designed for good readability in smaller sizes. We use Verdana as an alternative system font. It’s only used in the cases where it isn’t possible to use the profile font, for example digital documents and templates that are shared with people outside of Elvia who might not have the profile font installed. Verdana is less dense than Red Hat, but to simulate Red Hat in the best way you can pinch the text by reducing the letter spacing.
Titles (lg, md, sm, xs) are used as headings for your content. They will stand out from the rest with their heavyweight and enhance visual hierarchy. We have 4 headings in different sizes.
Title Caps can be used as a small subheading. Caps will not stand out the same way as titles but will enhance visual hierarchy together with text medium or small.
Text Lead is our biggest text option and used for introduction of an article between heading and paragraph as a brief summary of the content.
Text Large is our main option for longer text in articles at external webpages.
Text Medium is our main option for both long and short text in internal applications.
Text Small is the smallest variant of body text and can be used in addition to text large and medium.
Text Quote is only used when you have quotes in articles.
Text Image is only used together with an image for caption to provide more context.
Text Micro is used as small descriptive text such as dates and values. Should not be used in long paragraphs, as it is very small.
All typography can be used through a set of classes. The classes set multiple attributes, like font-size, line-height and font-weight. These are all available when using Elvis.

To use the CSS typography classes without using Elvis, you need to import the @elvia/elvis-typography package itself. It only provides SCSS-files, not CSS.
