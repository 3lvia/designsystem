User Feedback There’s a number of tools we use recurringly to analyze behaviour and test our solutions on real users. Whether its for user research or testing solutions, we have summarized a list of the most popular tools in Elvia below.
  1. What? Teston is an online solution for remote user testing. It recruits users and enables unmoderated testing over video.
  2. Why? We use Teston for continous user testing, especially when we need to test quickly, or when it’s harder to test on real users.
  3. How? If you have questions, or wonder if it could be useful in your project, reach out in our slack #designsystemet .
  1. What? HotJar is a Website Heatmaps & Behavior Analytics Tools. It collects user feedback on how visitors are using websites.
  2. Why? We mostly use HotJar for its chat and feedback functionality, as well as tracking how well our solutions perform in real time.
  3. How? If you have questions, or wonder if it could be useful in your project, reach out to Henriette Wiborg Sunde or in our slack #designsystemet .
  1. What? Google Analytics tracks and reports website traffic, such as visitors on the site, as well as information on the source.
  2. Why? We use mainly Google Analytics to track the number of visitors using our sites at a given time.
