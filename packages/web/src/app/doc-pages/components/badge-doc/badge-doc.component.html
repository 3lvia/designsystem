The badge component acts as a wrapper. You should only use badges on Icon Button or Thumbnails. Note how the badge places itself relative to its content, meaning you can place a badge component within other elements, such as the icon in an Icon Button element. Use slots to give the badge its content.
The green badge is the default, but use the color that suits your case best. Red can be used in cases where the case is negative, like deviations or missing information. The neutral (black/white) badge has a more neutral and toned-down expression.
Data-colors can also be used when you want to connect the badge to some data that uses a data-color already.
You can't rely on the badge's content being announced correctly. You should include a detailed description, such as aria-label .
