The spotlight should have a padding of approximately 24px from the element. To get the correct circle radius, find the largest measurement (width or height) of the object you want to light up, divide by 2, and add 24px to the find the correct radius. To find the correct radius you need to find the largest measurement, either height or width of the element and then you can use this formula:
radius = ((Math.max(width, height)/2) + 24px)
 To find the center of the element and therefore circle you can use these formulas:
horizontal = rect.left + rect.width / 2
vertical = rect.top + rect.height / 2
