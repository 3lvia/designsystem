Never use a language or word that the user does not understand. It should always be clear to the user what the actions are all about. Use short texts, up to 250 characters in body text. The modal requires implementation for closing. Use the callback prop 'onHide' to listen for click outside modal or on close btn and then set the 'isShowing' prop to false. Actions has to be implemented for all other buttons added to the modal as well.
