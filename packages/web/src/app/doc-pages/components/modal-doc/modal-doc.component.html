Modal content should be as short and concise as possible. The modal window is meant to catch the users attention for quick actions or short messages. Avoid comprehensive information, focus on relevance and information that can be scanned quickly be the users. You can add an illustration or image to the modal. The size of the modal then changes to fixed height and width (550 x 1090px). Illustration can be used with both Actions and Info modals. If you need multiple pages in the modal carousel are added to navigate. A close icon button is then added to make it easy to escape the modal. The Elvia modal will automatically trap the focus within the modal, and will by default focus on the first focusable element when opened. If you wish to customize what element is focused initially, you can use the elvisFocusInitial attribute on the element you want to give initial focus. The modal requires implementation for closing. Use the callback prop 'onClose' to listen for click outside modal or on close button and then set the 'isShowing' prop to false. Actions has to be implemented for all other buttons added to the modal as well.
Space or Enter to open or close the modal.
Esc to close the modal when open.
