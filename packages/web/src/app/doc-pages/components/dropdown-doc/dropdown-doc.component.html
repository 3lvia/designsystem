Figma sketches

This documentation shows how you should style your dropdown. We do not currently provide styling classes for this, because we understand that it is desirable to use libraries with dropdowns that have all the logic necessary, and works in an accessible way for all users. So in this documentation we have pictures of the design that you should try applying to your dropdown.

Take a look in figma to get the detailed view where you can inspect properties of the design.

To make the dropdown input field you might be able to use our autocomplete classes or input classes . These classes have the same styling as the dropdown input designed here. Make sure to asign the input type to "button" to not let user type inside the input field
