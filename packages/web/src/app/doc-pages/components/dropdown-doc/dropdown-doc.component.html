It's possible to display icons in the dropdown options. Simply add a key value pair of 'icon : iconName' to the dropdown options to create icons aligned on the left side of the label text.
Note that to display icons, all options need to have an icon attribute. Remember to pass icon names as camelcase, as under the hood we use the icon component to display icons within the dropdown component, click here for more information on the icon component
There are two different sizes of dropdown; normal and compact. Much like compact text fields, the compact dropdown is a version of the text field that’s primarily designed for internal systems, and shouldn’t be mixed with the other input style.
