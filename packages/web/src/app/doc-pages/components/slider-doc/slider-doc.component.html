The simple slider allows users to set and select a single value from a range. The range slider allows the user to specify a numeric range value that must be no less than a given value and no greater than another given value. The unit prop allows you to add a custom suffix to the input field(s) of the slider and the value shown in the tooltip of the slider. For example, if you want to display the value as a kilowatt-hour, you can add "kWh" as the unit.

The return value will not include the unit -string.
