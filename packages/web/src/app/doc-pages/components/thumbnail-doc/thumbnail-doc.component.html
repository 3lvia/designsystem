Use e-thumbnail---selected to visually highlight a selected thumbnail in a group.

Use e-thumbnail---hover to manually apply a green border around a thumbnail during hover. This also occurs automatically when a user hovers over the thumbnail in the browser.

This is an example of how a thumbnail should be used.
  • Aria label: Always add a descriptive aria-label to the thumbnail button.
  • Alternate text: Give the img an alt attribute explaining what the thumbnail button does.
