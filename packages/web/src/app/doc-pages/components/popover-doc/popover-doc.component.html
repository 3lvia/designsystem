In informative popover we offer popover as a shell with options like close button, title and text. Then you can add the content you need within the area - like text and inputs for example. Popover closes after clicking outside the popover, “X” or ESC KEY.
The title should be short and clear. The title is optional. You can have different type of content like text, buttons, input etc. The content will have 100% width, unless the content only consists of text, then the max-width is 450px. The trigger can be any kind of button. The popover will close if you click outside, ESC key, or the button connected to it. The close button is optional and can be removed in special cases like in a menu list.
