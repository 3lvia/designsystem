On mobile - You can put radio filter in a popover if there is a lack of space or if it should take up less focus. Popover is triggered by a button.
You can have an icon together with the text.
 Radio filters work like and built up of a radiobuttons, making the keyboard navigation identical to the keyboard navigation for native radiobuttons.
Space or Enter to select a radio filter.
