  • Role: Add role="tabpanel" to the corresponding content closeAriaLabel to specify for assistive tools what the accordion is showing and hiding.
  • Aria labelledby: Add aria-labelledby='ewc-tabs-0' to the corresponding contents, with the id of that tab. The id of the tab is named like this: 'ewc-tabs-' + index . NB! If you have multiple set of tabs you should use the tabIdPrefix and the id will then be 'ewc-tabs-prefix-index' e.g: 'ewc-tabs-fruits-3' .
  • Aria controls: The aria panels should also have unique id, which should correspond to the tabs aria-controls. The aria-controls are named in the same way that the ids for the tabs are named, so it should be easy to add the ids to panels e.g. id='ewc-tabs-fruits-3' .
