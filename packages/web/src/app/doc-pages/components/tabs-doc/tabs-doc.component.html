  1. Role:

    Add role="tabpanel" to each tabpanel (the element that contains the content associated with a tab).

  2. aria-labelledby:

    Add aria-labelledby="ewc-tabs-[index]" to each tabpanel.

  3. aria-controls:

    Add id="ewc-tabpanel-[index]" to each tabpanel.

Use the tabIdPrefix prop to avoid ID collisions when using multiple tab groups. IDs become ewc-tabs-[prefix]-[index] and ewc-tabpanel-[prefix]-[index]. e.g., tabIdPrefix="fruits" gives ewc-tabs-fruits-0 & ewc-tabpanel-fruits-0.
