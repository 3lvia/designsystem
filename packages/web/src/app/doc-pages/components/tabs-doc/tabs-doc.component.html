Tabs {{ description }}
Tabs with more than 6 options should be avoided and never fewer than 2 tabs. Keep the option label short, descriptive and unique and not longer than two words. Tabs in mobile will stay the same but work like a carousel (tabs with horizontal scroll). The tabs in the tabs-component should have corresponding content / tab-panels, where each element should have the attributes listed below:
  • role="tabpanel"
  • aria-labelledby="insertIdOfTab"
