Date Picker {{ description }}
There are two different sizes of the date picker; normal and compact. Much like compact text fields, the compact date picker picker is a version of the text field that’s primarily designed for internal systems, and shouldn’t be mixed with the other input style.
Space or Enter to open or close the datepicker, as well as choosing different options inside the datepicker.
Tab to navigate through the different options inside the datepicker.
can be used to choose between dates inside the datepicker.
The date picker component supports getting the selected date in two different ways: as a date object through valueOnChange , or as an ISO formated string (without containing time) through valueOnChangeISOString (e.g. '2022-06-20' ). When using the resetTime prop, the returned date object will contain the selected date, with the time set to 00:00:00 .
