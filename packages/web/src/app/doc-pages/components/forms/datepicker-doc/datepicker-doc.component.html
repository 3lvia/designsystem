Date picker {{ description }}
There are two different sizes of the date and time picker; normal and compact. Much like compact text fields, the compact datepicker and timepicker is a version of the text field that’s primarily designed for internal systems, and shouldn’t be mixed with the other input style. In the future there will be a date range picker component. For now combine 2 date picker components to make the wanted result. The from-date should have a max-date equal to the chosen to-date while the to-date should have a min-date equal to the chosen from-date.
