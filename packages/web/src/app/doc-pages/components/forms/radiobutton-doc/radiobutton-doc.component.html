There are two sizes of radio buttons: Normal and Compact. Make sure to always add a label that is linked to the radiobutton, to make it clear for assistive tools what the radiobutton is doing. Use the e-form-field classes to get correct styling of labels. The states have associated classes that can be used to programmatically add the desired styling. Remember to always add the state attribute ( checked , required , disabled , invalid and so on) to the input to inform assistive tools of the current state. Remember to add validation classes to help users understand what needs to be corrected. Read more about how the validation should happen .
