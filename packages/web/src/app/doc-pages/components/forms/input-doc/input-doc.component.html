Use the e-combined-field class on the form-field element to keep two inputs for the same label. Use the e-inlined-field class outside the form-field element to align inputs on the same line. When the zip code is filled in, the post office should be filled in automatically. The e-form-field__label--optional class should be added to the label when the input is not required. The view_on icon button should be used when showing and hiding a password. There are two different sizes of input. The width of the container should be proportional to the expected user input and can be changes by setting the width on the container of the input. Compact text fields are typically used in Elvia’s internal systems, where screen real estate is an important factor. Regardless of which style you use for text fields, you should always stick with the same type throughout the page and never mix the two. By default, all fields must be required. Validation is based on the field being filled out. Validation takes place along the way, and the user must also be presented with guidance or error messages.
