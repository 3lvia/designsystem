The Removable chip type is removed when it's clicked.
Legend chips should be used for filtering, primarily for visualization in graphs (e.g. colors representing different lines in a graph).
Choice chips can be used as a filter for multiple choice that you want the user to be able to toggle on and off.
The default color is green. It is the main color to use when there is only one category.
To categorize chips, different colors can be used to distinguish them from each other. All chips in the same color should be placed together. See the properties table at the bottom for all available colors.
