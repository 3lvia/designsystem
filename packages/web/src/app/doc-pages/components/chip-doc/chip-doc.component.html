Removable chip is used to show what filter the user have selected and can easily be removed by clicking on them.
Legend chips are a combination of a chip and a legend with multiple choice. It visualizes the color of the data you are toggling on/off in a graph or map.
Choice chips are always showing all options the user can select from and can be toggled on/off (multiple choice).
The default color is green. It is the main color to use when there is only one category.
To categorize chips, different colors can be used to distinguish them from each other. All chips in the same color should be placed together. See the properties table at the bottom for all available colors.
Space or Enter to select, unselect or delete a chip.
Tab to navigate through the chips.
  • Aria label: Remember to always supply a aria-label for the chip. It should also be updated whenever a chip is selected or unselected
