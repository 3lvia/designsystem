You can choose whether you want it to loop or not. In the last step, the right button then switches to a confirmation button. This can be used in a dialog for example, and the dialog will then close when clicking confirm button. If you want to have more control over the items, you can send just the number of items in to the carousel instead of a list of items/slots. This will only show the carousel with the correct number of steps. If you choose to use only the carousel, without sending in items, you have to add the animation for items sliding out and fading in yourself.
ArrowLeft/ArrowRight to navigate through the arrows and pages.
Space or Enter to select a carousel page.
