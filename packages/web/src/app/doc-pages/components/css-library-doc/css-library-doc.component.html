CDN Deprecated

We have deprecated the use of the CDN, and it is no longer recommended to use. This is because you miss out on automatic updates through Renovate. We recommend using NPM instead.

If you are still using the CDN, please note that the latest version available is 16.2.2. Newer versions will not be available through the CDN.

To migrate to NPM, first remove any old reference to Elvis from the CDN. This should be:
  • <link> to the elvis.css stylesheet in the <head> of your HTML file.
  • <script> to the elvis.js script at the bottom of your <body>.
Then follow the instructions in the NPM section above. To simplify the transition, it is possible to install the same version that was being used through the CDN.
