This package includes basic CSS-based components such as links, tags and inputs. These are all found in the list of components, and are marked with the tag “CSS Library” to indicate that they are used through this library. Different utilities are also available through this library. These include tokens like colors and typography, as well as assets like icons and logos. Npm is the recommended method of installing Elvis.
  1. Install package: {{ npmInstall }}

    or

    {{ yarnAdd }}
  2. Import stylesheet: {{ importStylesheet }}

    Add this import to your main stylesheet, for example "styles.scss".

  3. Import script: {{ importScript }}

    Please note that elvis.js should be imported only once, somewhere at the base-level of your app, for example "main.ts". Depending on what framework you are using, the method of doing this may differ.

The CDN can also be used. Note that this is not recommended, since you miss out on typings and automatic updates through Renovate.
