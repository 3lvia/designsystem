CSS Library The CSS library consists of multiple component classes and utilities like colors, icons and typography and are all available through @elvia/elvis .

The CSS library is its own package that provide the most basic utilities and components needed to create a new Elvia application.
To be able to use these components and utilities, all you need to do is install this package as described in the installation section and you are ready to go.

This package include all basic components like links, tags, simple inputs and many more. All components that are listed in the components menu together with the other components, but also marked with the label “CSS library”. A lot of utilities are also available through installing the package. These include tokens, like colors and typography, as well as assets, like icons and logos. Npm is the recommended method of installing Elvis.

  1. Install package:

    {{ npmInstall }}

    or

    {{ yarnAdd }}

  2. Import stylesheet:

    {{ importStylesheet }}

    Add this import to your main stylesheet, for example "styles.scss".

  3. Import script:

    {{ importScript }}

    Please note that elvis.js should be imported only once, somewhere at the base-level of your app, for example "main.ts". Depending on what framework you are using, the method of doing this may differ.

The CDN can also be used. Note that this is not recommended, since you miss out on typings and automatic updates through Renovate.

  1. Add stylesheet

    Head
    {{ linkTagCode }}

    This should be added to the head element.

  2. Add script

    Body
    {{ scriptTagCode }}

    This should be added to the body element.
