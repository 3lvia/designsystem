The progress bar is used to show loading, both locally and for full page actions where the Content loader isn't suitable. It is for example used in File Upload , where it shows progress or error for each file.
  • Aria label: Remember to always supply a aria-label for the progress bar. Use the ariaLabel prop.
  • Role: Use role="meter" when using the progress bar component as a gauge to improve accessibility for screen reader users. Use the ariaRole prop.
To improve the accessibility of a page with a loading progress bar, include the aria-describedby attribute on the region being loaded and set it to reference the progress bar's id (use the componentId prop). Set the aria-busy attribute on the region to true while it is loading.
