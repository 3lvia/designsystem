A toast will automatically be read by a screen reader in an assertive way. There are two steps necessary to use the toast component:
  1. Add the <elvia-toast></elvia-toast> component to the root of your app, for example App.vue (Vue), App.component.ts (Angular), or App.tsx (React). This is the actual toast component, and will listen for events dispatched by the openElviaToast method.
  2. Import the method openElviaToast from @elvia/elvis-toast where you want to dispatch a toast, and call it with the appropriate toast configuration. The different options for the toast configuration is displayed below.
