A toast will automatically be read by a screen reader in an assertive way. There are two steps necessary to use the toast component:
  1. Add the <elvia-toast></elvia-toast> component to the root of your app, for example App.vue (Vue), App.component.ts (Angular), or App.tsx (React). This is the actual toast component, and will listen for events dispatched by the openElviaToast method.
  2. Import the method openElviaToast from @elvia/elvis-toast where you want to dispatch a toast, and call it with the appropriate toast configuration.
The toast configuration has the following options:
  • title: string : The title showed in the toast.
  • body: string : The body showed in the toast.
  • duration: number : How long the toast is visible, in milliseconds. Defaults to 7000 .
  • closable: boolean : Whether the toast has a close button.
  • status: "positive" | "informative" : The toast status, which changes its appearance. Defaults to "positive" .
  • customIcon: string : Allows for writing inner HTML to replace the default icon. Example: <i class="e-icon e-icon--crane"></i>
