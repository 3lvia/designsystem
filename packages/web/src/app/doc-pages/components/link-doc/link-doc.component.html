Link
Standard links stand alone and direct you to another page. Inline link are used within sentences and text blocks, and will always carry the font size of the text block. New tab link is used when the page you are linking to opens in a new window or tab. You should open a link in a new tab when it leads outside of the website Action link are mostly used when you link to another page to do an action. It can also be used when you have a link alone and want it to take more attention than a regular link. Back link are used to navigate the user to a previous page or take the user to the start page. Back links have a green arrow next to it to make it stand out more and to clarify that you will be sent back. The e-btn class can also be used on anchor tags to make links appear like buttons. When needing a collection of links use either action group , a simple link list, or cards , when you want a more visual representation of content. Use meaningful labels for links and avoid terms like “click here”, and try to cut down the unnecessary words. Try to reflect the content user will find when clicking on the link.
