Link
Standard links stand alone and direct you to another page. Inline link are used within sentences and text blocks, and will always carry the font size of the text block. External link are used when the user are directed to a page not produced by Elvia. New tab link is used when the page you are linking to opens in a new window or tab. Action link are mostly used when you link to another page to do an action. It can also be used when you have a link alone and want it to take more attention than a regular link. Back link are used to navigate the user to a previous page or take the user to the start page. Back links have a green arrow next to it to make it stand out more and to clarify that you will be sent back. Jumbo link is used to create a clear overview of multiple links within the same category. The text content cannot exceed two desktop rows in the jumbo link. The desktop version will keep standard height while the mobile version will expand for text content according to the desktop limit. Card links have an icon, label, and alternatively a description text. The entire card link is clickable and are useful for grouping several related pieces of information together. Card links work best when they provide summaries of many different kinds of content to choose from. e-link--delay-hover if you want the ikon on hover to appear after scaling has happened. Standard link cards are the best to use when you have a lot of links, while the shortcut is better suited to fewer links. If 2 different icons are given the second will be visible on hover while the first is visible from start. You can add a description together with the label. Keep it short! The icon is scaling up and a green border appears when hovering. You can add an extra hover effect by switching a black icon with a two-colored icon. The size of the card link should be dynamic and responsive, but the card link should not be smaller than 112px wide and not larger than 400px wide. The e-btn class can also be used on anchor tags to make links appear like buttons. Use meaningful labels for links and avoid terms like “click here”, and try to cut down the unnecessary words.
