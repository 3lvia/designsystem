Normal will always have the same position above the content . Normal shows either none or all of the content. Overflow is placed below the content and moves down if expanded. Overflow shows some of the content and can be expanded to show all. Single are an accordion without label and can be placed next to other elements, for example in an list or table. To create an accordion group, use multiple accordions with isFullWidth=true grouped together, along with dividers and spacing.
