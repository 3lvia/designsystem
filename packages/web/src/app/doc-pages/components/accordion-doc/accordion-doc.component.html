Normal will always have the same position above the content . Normal shows either none or all of the content. Overflow is placed below the content and moves down if expanded. Overflow shows some of the content and can be expanded to show all. Single are an accordion without label and can be placed next to other elements, for example in an list or table. Group multiple accordions to make it easy for users to find relevant content faster. Let users know what to expect by making sure the accordion labels are descriptive. Think of the usefulness of a well-structured FAQ (Frequently Asked Questions). Read more on group patterns at our group documentation page . To create an accordion group, use multiple accordions with isFullWidth=true grouped together, along with dividers and spacing. If you want only one accordion to be open at a time, you can use the isOpen property along with the onOpen / onClose events to control the state of the accordions.
Space or Enter to open or close the accordion.
  • Aria label: Remember to use the props openAriaLabel and closeAriaLabel to specify for assistive tools what the accordion is showing and hiding.
