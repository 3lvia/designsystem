An object describing the heading and state of the steps should be configured. The keys of the object are the step indexes (starting at index 1), and the values are an object containing the heading and the state of each step index. The value object ( StepItem ) is described in the Stepper configuration section. To update the states with the steps prop, you have to make a new object. For example if you want to update step number 3 to become completed {{ 'newSteps = { ...oldSteps, 3: { isCompleted: true } }' }} .
ArrowLeft/ArrowRight to navigate through the steps.
Tab to navigate to the back/next buttons.
Space or Enter to select a step or button.
