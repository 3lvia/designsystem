In a forced stepper the user has to complete a step before continuing to the next one. They can always step backwards to an earlier step. If a field is not filled and the user tries to go forward an error message should appear. When there are more than 4 steps the stepper should collapse. Be cautious of adding more than 4 steps as a lot of steps can be overwhelming for the user.
