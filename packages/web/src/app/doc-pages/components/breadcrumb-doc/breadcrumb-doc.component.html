Breadcrumb {{ description }}
To use routing with the breadcrumb component, the callback function breadcrumbsOnChange can be used. Instead of passing in a URL to the breadcrumbs prop, the breadcrumbsOnChange callback function is provided with the index of the clicked breadcrumb. This functionality can be used to handle routing outside of the breadcrumb component allowing the use of routing modules (e.g. Angular RoutingModule or React Router).
