Breadcrumb {{ description }}
The default behaviour of the breadcrumb component is to make anchor tags with a given title and url from the breadcrumbs prop. This works, however it does not provide support for using routing modules to handle routing. When clicking a breadcrumb, the callback function breadcrumbsOnChange is called if provided, and the event breadcrumbsOnChange is dispatched. It is called with the index of the clicked breadcrumb as its only parameter. This functionality, combined with not sending in a url to the breadcrumb, can be used to handle routing outside of the breadcrumb component allowing the use of routing modules (e.g. Angular RoutingModule or React Router).
