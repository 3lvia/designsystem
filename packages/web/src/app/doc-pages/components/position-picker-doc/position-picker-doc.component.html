Position Picker {{ description }}
The position picker component consist mainly of container classes. Where each element is assigned for it intended purpose as icons, description text of picked position and action area for placing tertiary buttons. When open use a modal to open the position-picker. The design should follow the figma sketches shown here.
To implement the position-picker add logic for opening a modal when clicking the 'Add position' button. Then switch out the existing content with 'After position picked' design, when the position is picked.
