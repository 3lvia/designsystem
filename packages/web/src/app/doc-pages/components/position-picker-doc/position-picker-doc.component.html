The position picker component consist mainly of container classes. Where each element is assigned for it intended purpose as icons, description text of picked position and action area for placing tertiary buttons. When position is chosen replace the icon container with the description container along with the chosen position or information. In the working example above we have attached a modal with an iframe of google maps to the tertiary button within the position picker component. This is simply a visual illustration on how one could attach a service to the position picker component. It does not let you pick an actual position and is only meant to illustrate how an actual component should be presented. General styling guide here is to attach a modal containing a map service. Which in return updates the position picker with the chosen position. The information should then replace the icon container, and insert information in the description container.
