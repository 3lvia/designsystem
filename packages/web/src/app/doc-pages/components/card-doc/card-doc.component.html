Card Component coming soon!
You can add a description together with the header. Keep it short! The colored line can be used to categorize or to visualize a status. Do not use only one colored line on all cards or use without a purpose! The icon is scaling up and a green border appears when hovering. You can add an extra hover effect by switching a black icon with a two-colored icon. The size of the card link should be dynamic and responsive, but the card link should not be smaller than 112px wide and not larger than 400px wide. To implement the card component as a link it must be wrapped in a link or an anchor tag. To avoid hard refreshes of the webpage the links should use a router (e.g. Angular RouterModule or React Router).
