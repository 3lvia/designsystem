Use simple card when an icon together with a short title is enough information to describe the link. Select between an icon or a number as a visual item. A number is used if you need a counter. You can add a description together with the label. Usually only one word so keep it short! If you need a longer description use detail card instead. The colored line can be used to categorize or to visualize a status. Do not use only one color on all cards or without a purpose. Multiple cards with colored line can feel cluttered. The icon is scaling up and a green border appears when hovering. You can add an extra hover effect by switching a black icon with a two-colored icon. Use detail card when you need to give more information about the link. If you need to categorize the cards you can add a tag. You can add a corner icon if you need to add special information to some of the cards. For example: if some of the card links are locked and require access. To implement the card component as a link it must be wrapped in a link or an anchor tag. Remember to wrap the card inside a link element <a> to ensure correct HTML markup. Assign the correct heading level to the card heading using the headingLevel attribute. Do not skip one or more heading levels.
