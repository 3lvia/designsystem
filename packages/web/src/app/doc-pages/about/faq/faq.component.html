Yes. It's possible to use the design system's components in a Blazor WebAssembly app. You need to install the components via npm and use the components as web components together with ASP.NET Core Blazor JavaScript interoperability (JS interop).

Elvis – our style-only components – can be used as normal. On the get started page, you can read the step-by-step guide for getting started.

See Microsoft's ASP.NET Core Blazor JavaScript interoperability (JS interop) documentation for more details and up-to-date best practices for using JS in your Blazor app.
