 We have three different shadows; soft, medium and hard. The naming convention reflects the percentage of opacity on the base color of shadows.
Soft

The naming convention reflects the percentage of opacity on the base color of shadows.

All shadows can be used through a set of classes. The e-shadow-3 , e-shadow-6 and e-shadow-8 sets box-shadow blur and opacity. Softer shadows has higher blur and lower opacity, and harder shadows has lower blur, but higher opacity.

  • e-shadow-none
  • e-shadow-3
  • e-shadow-6
  • e-shadow-8

All shadows can be used through their CSS-variable as well.

  • var(--e-shadow-none)
  • var(--e-shadow-3)
  • var(--e-shadow-6)
  • var(--e-shadow-8)

If you have to support IE11 and want to use CSS-variables, please check out the get started section for how to: Get started

All CSS-variables can be used, except if you want remove shadow.
