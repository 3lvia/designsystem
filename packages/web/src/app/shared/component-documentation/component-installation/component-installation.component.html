Angular
Add CUSTOM_ELEMENTS_SCHEMA to the component module to enable support for web components.
schemas: [ CUSTOM_ELEMENTS_SCHEMA ]
  1. Install package:
    npm install @elvia/{{ componentData.name }}
  2. Import package:

    Web component

    Import '@elvia/{{ componentData.name }}';
    (Angular - Should be imported in module.ts not component.ts)

    React

    import { {{ componentData.elementNameR }} } from '@elvia/{{ componentData.name }}/react';
  3. Import types (optional):

    Types

    import { {{ componentData.elementNameR }}Props } from '@elvia/{{ componentData.name }}/react';
    Typescript interfaces for the component properties are available. They are exported from the React export, but can be used in any project.
