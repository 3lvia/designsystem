Angular
Add CUSTOM_ELEMENTS_SCHEMA to the component module to enable support for web components.
schemas: [ CUSTOM_ELEMENTS_SCHEMA ]
  1. Install package:
    npm install {{ packageName }}
    or
    yarn add {{ packageName }}
  2. Import package:

    Web component

    import '{{ packageName }}';
    (Angular - Should be imported in module.ts not component.ts)

    React

    import { {{ reactElementName }} } from '{{ packageName }}/react';
  3. Import types (optional):

    Types

    import { {{ reactElementName }}Props } from '{{ packageName }}/react';
    Typescript interfaces for the component properties are available.
