Popover
Popover is a non-modal dialog that appears over the content on the screen. Popover is used with a clickable trigger element and should position itself relative to where there is free space on the screen. You can close the popover either by clicking “x” or outside the component. The class e-popover---visible needs to be added to show the popover. The popup will align to the top left position by default.
View live example, written in TypeScript, at the bottom.
