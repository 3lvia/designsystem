Table
A table displays rows and columns with data and makes it efficient to look up and compare values. Text should be left aligned in the columns and numbers should be right aligned.
Tables are available in two levels of density (a larger table and a compact table) as well as in two types of styles (light header and dark header). The light version is best suited when using multiple tables on one page. When using tables, make sure to always use the same type of density and style on all tables throughout the page.
Note: Note: The Elvia design system does not include table functionality, which means the design for mobiles included here will not automatically appear when switching to smaller screens.

Table sorting

The sorting arrow appears when hovering over any of the sortable top tabs. When clicked, the table will sort according to the chosen column; either ascending (arrow up), descending (arrow down), or original order (no arrow). When active, the arrow will stay visible in the table header until it’s either deactivated or replaced by another column sorting.

Add the e-table__sort-header to the th-tag on the columns you want to sort. Then add the 2 icons for arrow up and down inside the span with class e-table__sort-icon . When toggling between arrow down and up, add or remove the class e-table__sort-icon--up . When toggling showing the arrow when active add or remove the class e-table__sort-icon--inactive .
