Local alerts are messages related to the site and the content. It can be placed anywhere between the header and footer, but close to the content it is related to. Local messages can also be used for feedback after a user actions. The maximum width is 850px. The status of an alert helps to communicate the information and the severity of the alert. Each status consists of a defined color and an icon. It is important to use the correct status of the alert to provide a consistent and universal user experience.
