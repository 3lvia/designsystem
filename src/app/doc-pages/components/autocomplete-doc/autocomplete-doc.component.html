Autocomplete Autocomplete fields can be used instead of long dropdowns. For example filling out which country you reside in. Instead of the user having to scroll down a list of countries, users can fill out the autocomplete field and the the valid options are filtered. If a user types something that does not correspond to any choices, an error message will appear.
