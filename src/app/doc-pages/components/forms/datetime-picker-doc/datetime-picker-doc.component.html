Date & Time picker
This documentation shows how you should style your date and time picker. We do not currently provide styling classes for this, because we understand that it is desirable to use libraries with date/time pickers that have all the logic necessary. So in this documentation we have pictures of the design that you should try applying to your date/time picker.
Take a look in figma to get the detailed view where you can inspect properties of the design.

Designing from scratch

If you need to style the datepicker or timepicker from scratch you can use our utilities classes which provide styling that can be useful.
  • typography
  • spacing
  • shadows
  • colors

Design timepicker

To make the timepicker you might be able to use our autocomplete classes or input classes. These classes have the same styling as the timepicker designed here.

Design datepicker

To make the datepicker you might be able to use our input classes.
To make the opened datepicker you might be able use some styling we provide in the autocomplete classes. But it is probably necessary to style from scratch.
