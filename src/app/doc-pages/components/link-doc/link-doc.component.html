Link
Links are navigational elements that take the user to a new page or context. Links can be used alone or inline with text.

Link or button?

Link takes you to another place while button does something. Don’t use button for navigational actions.
Use Button: Actions where users affect the website’s back-end or front-end
Use Link: Actions where users won’t affect the website at all.

Label

Use meaningful labels for links and avoid terms like “click here”, and try to cut down the unnecessary words. Try to reflect the content user will find when clicking on the link.
Do: Se vårt strømsbruddskart
Dont: Klikk her
