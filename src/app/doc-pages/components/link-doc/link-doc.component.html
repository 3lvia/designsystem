Link
Links are navigational elements that take the user to a new page or context. Links can be used alone or inline with text.

Link or button?

Link takes you to another place while button does something. Don’t use button for navigational actions.
Use Button : Actions where users affect the website’s back-end or front-end
Use Link : Actions where users won’t affect the website at all.

Label

Use meaningful labels for links and avoid terms like “click here”, and try to cut down the unnecessary words. Try to reflect the content user will find when clicking on the link.
Do:  Se vårt strømsbruddskart
Don't:  Klikk her

Standard vs shortcut

Standard link cards are the best to use when you have a lot of links, while the shortcut is better suited to fewer links.

Icons

If 2 different icons are given the second will be visible on hover while the first is visible from start.

Description

You can add a description together with the label. Keep it short!

Hover

The icon is scaling up and a green border appears when hovering. You can add an extra hover effect by switching a black icon with a two-colored icon.

Dynamic size

The size of the card link should be dynamic and responsive, but the card link should not be smaller than 112px wide and not larger than 300px wide.
