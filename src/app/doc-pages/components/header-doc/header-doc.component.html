Header
Headers are used for navigation on the website and should be displayed at the top of the page. There are two types of headers available: one for external(coming) and one for internal systems.
Internal - Preview
NB! This preview is not a finished working header.
This header only shows some logic to visualize how it should look. Logic has to be applied for opening, closing and choosing menus and items.
  • The modifiers classes listed below should be added and removed when clicking menus and choosing options.
  • When clicking outside of any menu or if choosing any option the menus should close.
Internal - Active & Open modifiers
To open a menu in the top-bar one or more of the following classes needs to be added.
New class Where When
e-top-bar__app--open e-top-bar-desktop When opening app-menu
e-top-bar__area--open e-top-bar-desktop When opening area-menu
e-top-bar__user--open e-top-bar-desktop When opening user-menu
e-sidebar--open e-sidebar When opening side-menu
e-header--open e-header When opening top-menu on mobile
e-top-bar-mobile__app--open e-top-bar-mobile When opening app-menu on mobile
e-top-bar-mobile__area--open e-top-bar-mobile When opening area-menu on mobile
e-top-bar__app-circle--active e-top-bar__app-circle Should always be present on 1 element
e-sidebar__item--active e-sidebar__item Should always be present on 1 element
