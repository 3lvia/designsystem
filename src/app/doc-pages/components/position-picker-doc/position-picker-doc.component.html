Position Picker Position selector lets you select a position and gives you coordinates. You can choose either by clicking on a map or text input field.

Visual example of position picker

In this example we have attached a modal with an iframe of google maps to the tertiary button within the position picker component.

General styling guide here is to attach a modal containing a map service. Which in return updates the position picker with the chosen position. The information should then replace the icon container, and insert information in the description container.

The example below is simply a visual illustration on how one could attach a service to the position picker component. It does not let you pick an actual position and is meant purely as an illustraion on how an actual component should be presented.
