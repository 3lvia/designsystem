Method 1: Easiest

Download the npm package and use Elvis.js from inside the npm package like this:
<script src="path_to_file/elvis.js"></script>

Elvis npm package

The icons can then be used like this: class="e-icon e-icon--chat e-icon--md"

How it works:
Elvis.js looks for e-icon and similar classes and injects a img tag with the SVG icon at the desired location.

Method 2: Using JS Module / SVGs

All icons are accessible as SVGs within a JS module. Import the required icon individually where you need them:

import {{ '{' }} addCircle {{ '}' }} from '@elvia/elvis/icons'

To get the icon you can use the following: addCircle.getIcon()

If you want a inverted icon use the following: addCircle.getIcon('inverted')

If you want a color: addCircle.getIcon('white')

Or set the color yourself with a hex: addCircle.getIcon('#bdbdbd')
