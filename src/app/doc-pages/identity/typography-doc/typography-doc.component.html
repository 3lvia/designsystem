Typography
Red Hat is Elvia's profile font and should be used throughout all material. The font family has two optical sizes, Display and Text.
All typography can be used through a set of classes. The font-classes sets multiple attributes, like font-size, line-height and font-weight.

Using Classes

All typography can be used through a set of classes. The e-title-lg sets multiple attributes, like font-size, line-height and font-weight.

Title classes

  • e-title-lg
  • e-title-md
  • e-title-sm

Text classes

  • e-text-lead
  • e-text-quote
  • e-text-body
  • e-text-img
  • e-text-description
  • e-text-caps
  • e-text-info

Label classes

  • e-text-label
  • e-text-option
