Red Hat Display has denser letter space and higher x-height and is suitable in large sizes, for example in titles. Red Hat Text has a slightly smaller x-height, more space and is designed for good readability in smaller sizes.

Red Hat was originally commissioned by Pentagram and was designed by Jeremy Mickel, MCKL. It is an open-source font family and you can download Red Hat here , via Google Fonts.

As an alternative system font, we use Verdana . It’s only used in the cases where it isn’t possible to use the profile font, for example digital documents and templates that are shared with people outside of Elvia, who might not have the profile font installed. Verdana is less dense than Red Hat, but to simulate Red Hat in the best way you can pinch the text by reducing the letter spacing.

All typography can be used through a set of classes. The font-classes sets multiple attributes, like font-size, line-height and font-weight.
