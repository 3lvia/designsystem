Grid This grid-system is based off of Bootstraps grid-system and works almost the same. Read the Bootstrap documentation here . All the Bootstrap grid-system classes are available for use. To use the grid-system add the e-grid class on a wrapper element. The bootstrap-classes will only work inside this wrapper, and will therefore not break any other potential grid-systems.

Breakpoints for different devices

The grid-system follows a 12 column rule, but the max number of columns should not exceed 8 on tablet and 6 on mobile.
Device Sizes Max number of columns
Mobile -> 767px 6
Tablet 768px -> 1023px 8
MinDesktop 1024px -> 1439px 12
MaxDesktop 1440px -> 12
