Grid {{ description }} Read the Bootstrap documentation here .
A grid layout for external and internal systems. Use the class e-grid-gutters-ext (for external systems) or e-grid-gutters-int (for internal systems) to add dynamic gutters on rows. Get vertical gutters as well by adding the class e-grid-gutters-vertical. Use the class e-grid-margin-ext (for external systems) or e-grid-margin-int (for internal systems) to add dynamic margin on rows. Combine the padding classes (eks. px-lg-3) and negative margin classes (eks. mx-lg-n3 ) to add your own custom gutters. Use no-gutters to remove gutters on the row. Use col-sm, col-md and col-lg to change grid at breakpoints. Use justify-content, align-content, and offset to align col-classes inside grid. Use order to choose the order of columns. The grid can have levels.

The grid-system follows a 12 column rule, but the max number of columns should not exceed 8 on tablet and 6 on mobile.

Device Sizes Max number of columns
Mobile -> 767px 6
Tablet 768px -> 1023px 8
MinDesktop 1024px -> 1439px 12
