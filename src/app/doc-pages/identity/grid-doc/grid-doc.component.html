Use col-sm, col-md and col-lg to change grid at breakpoints. A grid layout for external and internal systems. If you donâ€™t need to use all 12 column individually, you can group the columns together to create wider columns. Use justify-content, align-content, and offset to align col-classes inside grid. Use order to choose the order of columns. The grid can have levels. Use the class e-grid-gutters-ext (for external systems) or e-grid-gutters-int (for internal systems) to add dynamic gutters on rows. Get vertical gutters as well by adding the class e-grid-gutters-vertical. Combine the padding classes (eks. px-lg-3) and negative margin classes (eks. mx-lg-n3 ) to add your own custom gutters. Use no-gutters to remove gutters on the row. Use the class e-grid-margin-ext (for external systems) or e-grid-margin-int (for internal systems) to add dynamic margin on rows.
