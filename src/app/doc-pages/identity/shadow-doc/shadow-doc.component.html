Soft, medium and hard

We have three different shadows; soft, medium and hard. Soft are mostly used on grey background, while medium and hard should be used on white background. The naming convention reflects the percentage of opacity on the base color of shadows.

Using Classes

All shadows can be used through a set of classes. The e-shadow-3 , e-shadow-6 and e-shadow-8 sets box-shadow blur and opacity. Softer shadows has higher blur and lower opacity, and harder shadows has lower blur, but higher opacity.

  • e-shadow-none
  • e-shadow-3
  • e-shadow-6
  • e-shadow-8

Using CSS-variables

All shadows can be used through their CSS-variable as well.

  • var(--e-shadow-none)
  • var(--e-shadow-3)
  • var(--e-shadow-6)
  • var(--e-shadow-8)

Do's and Don'ts

All CSS-variables can be used except if you want remove shadow.
