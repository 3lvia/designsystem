 The spacing classes e-p and e-m styles the padding and margin attributes and can be specified to style left, right, top or bottom. The spacing system follows multiplication 8px.

All spacing can be used through a set of classes. The e-p-64 sets padding to the size of the last number. The padding and margin sizes are multiplications of 8px, meaning e-m-24 equals margin: 24px . The classes are available from 0 to 72.

Padding classes

  • e-p-0
  • e-p-number
  • e-pl-number
  • e-pr-number
  • e-pt-number
  • e-pb-number
  • e-px-number
  • e-py-number

Margin classes

  • e-m-0
  • e-m-number
  • e-ml-number
  • e-mr-number
  • e-mt-number
  • e-mb-number
  • e-mx-number
  • e-my-number
  • All the margin classes also have negative classes by adding 'n' before number (eks. e-mr-n8)

All spacing can be used through their CSS-variable as well. To use the CSS-variables you need to install the ponyfill css-vars-ponyfill. Click here more information about how.

  • var(--e-spacing-number)
