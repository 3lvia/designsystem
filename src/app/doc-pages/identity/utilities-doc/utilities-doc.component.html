Utility Classes
Elvis offers several utility classes that can be applied on elments without making any extra CSS classes or modifications in your project
Table of classes
Class name Use case CSS props
e-none Remove elements from the DOM display: none
e-invisible Hide elements in DOM visibility: hidden
e-capitalize Transforms the first character of each word to uppercase text-transform: capitalize
e-lowercase Transforms all characters to lowercase text-transform: lowercase
e-uppercase Transforms all characters to uppercase text-transform: uppercase
