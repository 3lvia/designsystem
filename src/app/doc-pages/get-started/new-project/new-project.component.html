New project with Elvis

What is Elvis?

Elvis is the design system Elvia use for external and internal systems. It helps designers and developers create software with a consistent look and feel. Elvis also saves time by allowing us to reuse components and design elements across multiple systems and sketches.

Contact / Support

If you have any issues at all, please don't hesitate to contact us! Any friction you experience in the onboarding process, or during use is valuable feedback for us

Get started (Using NPM)

1. Install the package:

npm i @elvia/elvis

2. Import main.scss in your stylesheet:

@import '~@elvia/elvis/src/main.scss';

3. Add elvis.js from the NPM package in <head> or <body>

Optional: Using CSS-variables

For most of the identity like colors, shadows and spacing CSS-variables are also provided. All these variables are available when importing the main.scss file in your stylesheet, meaning you don't need to import any specific files to use them.
The syntax for CSS-variables: var(--e-name-of-class)
