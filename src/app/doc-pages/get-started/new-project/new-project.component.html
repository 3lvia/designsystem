Optional: CSS-variables

Sometimes creating your own classes is necessary. For most of the identity like colors, shadows and spacing CSS-variables are also provided. All these variables are available when importing the main.scss file in your stylesheet, meaning you don't need to import any specific files to use them.
The syntax for CSS-variables: var(--e-name-of-class)
