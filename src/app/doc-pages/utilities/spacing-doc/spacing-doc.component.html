Using Classes

All spacing can be used through a set of classes. The e-p-1 sets padding to the size of the last number. The padding and margin sizes are multiplications of 0.5rem (8px), meaning e-m-3 equals margin: 24px. The classes are available from 0 to 9.

Padding classes

  • e-p-0
  • e-p-number
  • e-pl-number
  • e-pr-number
  • e-pt-number
  • e-pb-number

Margin classes

  • e-m-0
  • e-m-number
  • e-ml-number
  • e-mr-number
  • e-mt-number
  • e-mb-number

Using CSS-variables

All spacing can be used through their CSS-variable as well.

Variables

  • var(--e-spacing-number)
