Shadow Shadows are used on elements to create depth. Which shadow variable to use is dependable on context and use. Guidelines here are to use soft shadow on grey background, while medium or hard shadows are to be used on white backgrounds.

Shadows

There are 3 shadows available in the Elvis. The naming convention reflects the percentage of opacity on the base color of shadows.

Using Classes

All shadows can be used through a set of classes. The e-shadow-3,e-shadow-6 and e-shadow-8 sets box-shadow blur and opacity. Softer shadows has higher blur and lower opacity, and harder shadows has lower blur, but higher opacity.

Classes

  • e-shadow-none
  • e-shadow-3
  • e-shadow-6
  • e-shadow-8

Using CSS-variables

All shadows can be used through their CSS-variable as well.

Variables

  • var(--e-shadow-none)
  • var(--e-shadow-3)
  • var(--e-shadow-6)
  • var(--e-shadow-8)

Do's and Don'ts

All CSS-variables can be used except if you want remove shadow.
