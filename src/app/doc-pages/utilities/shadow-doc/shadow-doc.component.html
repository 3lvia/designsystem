Shadow Shadows are used on elements to create depth. Which shadow variable to use is dependable on context and use. Guidelines here are to use soft shadow on grey background, while medium or hard shadows are to be used on white backgrounds or cards.

Shadows

Using Classes

All shadows can be used through a set of classes. The e-shadow-1 sets box-shadow to colors from lightest to darkest depending on the last number. This means, the code e-shadow-3 equals box-shadow: 0 2px 4px rgba(0, 0, 0, .5).

Classes

  • e-shadow-none
  • e-shadow-1
  • e-shadow-2
  • e-shadow-3

Using CSS-variables

All shadows can be used through their CSS-variable as well.

Variables

  • var(--e-shadow-number)

Do's and Don'ts

All CSS-variables can be used except if you want remove shadow.
