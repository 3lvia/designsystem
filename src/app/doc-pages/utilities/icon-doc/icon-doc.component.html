Using Classes

All icons can be used through a set of classes. The e-icon-name, where 'name' is the name of the icon, which are all listed in the overview, can be used in an i element to display icons. The code e-icon-clock will show a clock. All icons can be sized by using the five sizing classes from extra large to extra small.

Icon classes

  • e-icon-name

Size classes

  • e-icon-lg

  • e-icon-xs
