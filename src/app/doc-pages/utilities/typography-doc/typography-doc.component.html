Using Classes

All typography can be used through a set of classes. The e-title-large sets multiple attributes, like font-size, line-height and font-weight.

Title classes

  • e-title-large
  • e-title-medium
  • e-title-small

Text classes

  • e-text-lead
  • e-text-quote
  • e-text-caps
  • e-text-body
  • e-text-description
  • e-text-info

Label classes

  • e-label-input
  • e-label-option
