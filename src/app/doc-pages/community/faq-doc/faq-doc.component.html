How do I report bugs?

Come in contact with us with any requests or reports of bugs and issues on slack #designsystemet . If you are a developer at Elvia you can also send requests or report bugs and issues through this github form . If these methods are unavailable to you, send a mail to michael.pande@eidsiva.no .

Which browsers are supported?

Components are actively tested for these browsers and only minor visual differences should occur:

  • Chrome - latest
  • Firefox - latest
  • Safari - latest
  • MS Edge - latest
  • MS Internet Explorer - 11
